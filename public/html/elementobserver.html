Change Tracker

  • This directive is used to watch over the value changes of form elements based on user interactions.
  • It has the ablility to watch over the values and execute a dynamic function provided through options.
  • This directive is implemented to have more developer control over value changes rather than using $scope.$watch


Script :

Add the following change-tracker directive to your Form control and provide the options in your controller.(Go through below samples.)

<input type="checkbox" change-tracker="checkOptionsCt" value="Checkbox1" ng-model="ctrlModel.checkbox1" name="checkboxTest" />
<input type="text" change-tracker="textOptionsCt" ng-model="ctrlModel.textbox" />
$scope.checkOptionsCt = {
    defaultValue: true,
    onTrackerChange: function (isModify) {
	    // Code to execute
    }
}

$scope.textOptionsCt = {
    defaultValue: "This is a test",
    eventType: 'keyup',
    onTrackerChange: function (isModify) {
	    // Code to execute
    }
}

$scope.commonOptionsCt = { 
    defaultValue: { 
        textType: "This is a test", 
        radioTypeGroup: "radio2", 
        checkType: false 
    }, 
    onTrackerChange: function (isModify) { 
	    // Code to execute
    }, 
    isGroup: true 
}

Parameters :

Parameters Description
change-tracker Following options are required:
  • defaultValue: default value for the control (onTrackerChange will get executed if the default value changes)
  • eventType: event type to bind (Optional)
  • isGroup: boolean to let the tracker know if it needs to look for a group of controls. If 'true' control name should match with the defaultValue property name
  • onTrackerChange: callback function if the value changes
name A unique name for the control.
